<h3>Name</h3>
<br>
・バンドリの中の人を呼び出すC#のコード

<h3>Overview</h3>
<br>
・bangdream.cs
→C#が記述された.csファイルです。
<br>
・bangdream.exe
→「mono」というコンパイラを使用して実行できます。

<h3>実行手順</h3>
※MacOS、ターミナルにて
<br>
$ brew install mono
<br>
$ mono bangdream.exe


.csファイルに変更を加えた際は、

$ mcs bangdream.cs

で適宜更新してください。
